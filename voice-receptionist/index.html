Schnellstart
Konfiguration
Deployment
1

Repository klonen

Server-Code herunterladen und Dependencies installieren.

git clone [repository-url]
cd voice-receptionist/server
npm install
2

API-Schlüssel einrichten

Kopiere .env.example zu .env und trage deine Keys ein.

cp .env.example .env
# Dann bearbeiten:
TWILIO_ACCOUNT_SID=ACxxx...
ANTHROPIC_API_KEY=sk-ant-...
ELEVENLABS_API_KEY=xxxx...
BUSINESS_NAME=Mein Salon
BUSINESS_TYPE=friseur
3

Server starten

Entwicklungsserver mit ngrok für lokale Tests starten.

npm run dev
# In zweitem Terminal:
ngrok http 3000
4

Twilio Webhook setzen

In der Twilio Console bei deiner Rufnummer:

Incoming Call Webhook:
https://[deine-url]/voice/incoming

Status Callback:
https://[deine-url]/voice/status
5

Anrufen & testen!

Rufe deine Twilio-Nummer an. Lisa meldet sich automatisch.
